## Summary
A backend API modeling MongoDB and wrappering github request.

## Installation

```bash
npm install
```

or, if you use [yarn](https://yarnpkg.com/) as package manager:

```
yarn
```

## Using

```bash
nodemon app.js
```

Look at :8080 port!
